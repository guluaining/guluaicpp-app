## 1. 执行摘要 (Executive Summary)

**GuluCoding 的 MVP 成功只是冰山一角。** 我们的终极愿景不是做一个C++教学软件，而是构建**“Gulu Edu-OS”**——一套通用的、AI驱动的 K12 互动教学操作系统。

我们将采用**“矩阵式战略 (Matrix Strategy)”**：
*   **纵向（学段）：** 覆盖 Pre-K 至 G9（从启蒙到竞赛）。
*   **横向（学科）：** 以 C++ 为切入点，快速横向扩展至 ESL（英语）、早期数学（Math）、科学（Science）。
*   **商业模式：** “麦当劳式”ToB 赋能（系统+内容+SOP）与 “Duolingo式”ToC 自增长双轮驱动。

我们的核心壁垒在于**“产研解耦，AI 生成”**：通过自研的通用互动引擎，配合 AI Vibe Coding 技术，实现内容生产的指数级降本增效。

## 2. 核心技术架构：产研解耦 (Decoupled Architecture)

这是我们能够快速扩科、降低成本的关键。

### 2.1 传统模式 vs. Gulu 模式
*   *传统模式：* 做数学课招数学开发团队，做英语课招英语开发团队。系统烟囱式林立，维护成本极高。
*   **Gulu 模式 (Edu-OS):**
    *   **底层：通用互动引擎 (Universal Interactive Engine)** —— 由**系统开发组**负责。它不懂数学或英语，但它懂“拖拽”、“匹配”、“填空”、“动画触发”、“AI 对话”、“积分系统”。
    *   **上层：内容配置文件 (Content Configs)** —— 由**内容开发组** + **AI** 负责。

### 2.2 AI Vibe Coding 内容生产管线
我们将引入 AI 辅助编程 (Vibe Coding) 彻底改变内容生产：

1.  **教研设计 (Human):** 资深老师设计课程脚本（如：G2数学-加法进位）。
2.  **AI 生成 (AI Agent):** 内容开发者使用 AI 工具，输入自然语言脚本，AI 自动生成适配 Gulu 引擎的 JSON 配置文件或轻量级逻辑代码。
    *   *Prompt:* "Create a drag-and-drop game for 2-digit addition with carry, creating visual bubbles like our C++ variable boxes."
3.  **引擎加载 (System):** 引擎读取配置，瞬间渲染出数学游戏。
4.  **人工审核 (QA):** 确保教育准确性。

**结论：** 技术团队维护一套引擎，内容团队（老师）在 AI 帮助下可以生产所有学科的 App。

## 3. 商业模式 (Business Model)

### 3.1 To B: “麦当劳”式整店输出 (The Franchise Model)
我们不只是卖软件，我们卖的是**开学校的能力**。
*   **Target:** 线下培训机构、私立学校、想要转型的教培中心。
*   **Offering (SaaS + Content + Service):**
    *   **中央厨房 (SaaS):** 选课、排课、学生管理、AI 批改系统。
    *   **料理包 (Content):** 标准化的 Coding/ESL/Math 互动课程体系。
    *   **SOP (Service):** 教师培训手册、课堂管理流程、家长反馈模板。
*   **Revenue:** 系统年费 + 账号授权费 + 教师培训费。

### 3.2 To C: 免费增值与自适应学习 (Freemium & Self-Paced)
*   **Target:** 全球 C 端家庭用户。
*   **Model:** 类似于 Duolingo/ABCya。
*   **Revenue:** 
    *   Free: 基础内容（带广告或限制体力）。
    *   Premium Subscription: 解锁所有学科、无限 AI 辅导、离线下载、详细成长报告。

### 3.3 Hybrid: 在线真人/双师课堂 (Online Live Class)
*   **Model:** 使用 Gulu 系统进行的真人在线授课（高客单价）。
*   **Value:** 解决家长“没时间管”和“需要真人情感连接”的需求。

## 4. 第一阶段执行路线图 (Phase 1 Roadmap: 6-12 Months)

**目标：** 跑通“通用引擎”，并占领 Coding + ESL + Early Math 三个高地。

**Q1-Q2: 引擎固化与 C++ 深耕**
*   **Tech:** 完成 Next.js + Supabase 全栈重构。开发“通用题目编辑器”。
*   **Content:** 完善 C++ G3-G6 课程。
*   **Biz:** 签约 5-10 家种子机构（B端），验证 SOP。

**Q3: 扩科验证 (ESL & Math)**
*   **Tech:** 引擎升级支持“语音评测 API”（为 ESL）和“手写识别/公式渲染”（为 Math）。
*   **Content (AI Powered):** 
    *   启动 **ESL (PreK-G3):** Phonics, Sight Words, Spelling Bee 游戏化课程。
    *   启动 **Math (G1-G6):** 可视化算术。
    *   *利用 AI Vibe Coding 将内容生产效率提升 500%。*
*   **Biz:** 推出“Gulu 全科会员”，提高 LTV（用户生命周期价值）。

**Q4: 全球化与规模化**
*   **Ops:** 正式拆分中美运营团队。中国区申请 ICP 经营许可证。
*   **Sales:** 启动 ToB 渠道代理商招募。
